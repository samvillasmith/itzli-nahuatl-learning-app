# Eastern Huasteca Nahuatl (EHN) Vocabulary — Exhaustive Revision Notes

**Purpose:** Line-by-line audit of the 837-entry primer vocabulary list for accuracy as Eastern Huasteca Nahuatl (ISO 639-3: nhe), cross-referenced against IDIEZ dictionary entries, Karttunen's Analytical Dictionary of Nahuatl, Wiktionary (nhe), the Huasteca Nahuatl Wikipedia article, and attested EHN texts.

**Orthographic standard used for corrections:** IDIEZ (ACK norm) — the most widely used scholarly orthography for EHN. Where INALI or SEP variants are also common, they are noted.

**Key to revision codes:**
- ✅ = Correct as given (form and definition attested for EHN)
- ⚠️ = Partially correct — needs minor fix (spelling variant, imprecise gloss, etc.)
- ❌ = Incorrect — wrong definition, wrong language variety, or fabricated entry
- 🔄 = Duplicate of another entry (noted but not necessarily an error)
- 📝 = Note on orthographic variant (entry is valid in a different orthographic system)

## UNIT 01 — Alphabet / Phonology

| Entry | Status | Notes |
|-------|--------|-------|
| a — vowel [a] like 'a' in father | ✅ | Correct |
| e — vowel [e] like 'e' in bed | ✅ | Correct |
| i — vowel [i] like 'ee' in feet | ✅ | Correct |
| o — vowel [o] like 'o' in go | ⚠️ | EHN /o/ is typically mid-low [ɔ], not as high as English "go". Better comparison: 'o' in "more" or "thought" |
| ā ē ī ō — long vowels | ⚠️ | Correct concept. Note: macrons are used in IDIEZ orthography but are frequently omitted in everyday writing. Long vowels are phonemic in EHN. Example word "piālli" is itself not standard — the common greeting is "pialli" (without macron in most sources) |
| h — glottal stop | ⚠️ | In EHN (Chicontepec variety), /h/ is typically realized as [h] (a breathy aspiration), not a true glottal stop [ʔ] as in Classical Nahuatl. The description "silent at word-end" is inaccurate for EHN — it is audible in careful speech. Some orthographies use ⟨j⟩ instead (SEP, SIL) |
| ch — [tʃ] | ✅ | Correct |
| tl — [tɬ] | ✅ | Correct. The description "tip of tongue on upper teeth, air released sideways" is a reasonable approximation |
| tz — [ts] | ✅ | Correct |
| x — [ʃ] | ✅ | Correct |
| hu / uh — [w] | ✅ | Correct for IDIEZ orthography. INALI uses ⟨w⟩ instead |
| qu / cu — [k] | ⚠️ | Correct for IDIEZ. More precisely: qu before e/i, c before a/o (not "cu before a/o" — cu represents /kʷ/ i.e., labialized k). The entry conflates /k/ (written c/qu) with /kʷ/ (written cu/uc). INALI uses ⟨k⟩ for both |
| z — [s] | ⚠️ | Correct for IDIEZ (z before a/o, c before e/i = [s]). INALI/SEP use ⟨s⟩ throughout |
| l m n p t y | ✅ | Correct — similar to English counterparts |

**Unit 01 general note:** The phonology section is reasonably accurate but glosses over the EHN-specific realization of /h/ and conflates c/qu with cu/uc. A note about which orthographic system is being taught should be added.

## SUMMARY OF CRITICAL ERRORS

### Definitions that are flat-out WRONG:

1. **Unit 10: "yankuik" = "bad"** → Actually means **"new"** (yancuic/yankuic)
2. **Unit 06: "xolotl" = verb "to be afraid"** → Xolotl is a **noun** (deity name/dog). The verb "to be afraid" = mahui/momauhtia
3. **Unit 06: "aqui" = "to be (negative)"** → "Aqui" means **"to enter"**
4. **Unit 06: "xawa" = "to be fashion"** → Nonsensical. "Xahua" means **"to apply face paint"** or **"to dig/excavate"**
5. **Unit 07: "citlal cetl" = "snow"** → "Citlalli" = star, "cetl" = ice. Snow = **"cepayahuitl"**
6. **Unit 09: "huehue" = "husband"** → "Huehue" means **"old man/elder"**
7. **Unit 02: "quema" = "when?"** → "Quema(h)" means **"yes"**. "When?" = **"queman"**
8. **Unit 23: "acalli" = "stilt house"** → "Acalli" means **"canoe/boat"**
9. **Unit 31: "tecotli" = "god"** → Not standard. "God" = **"teotl"**
10. **Unit 32: "yacametl" and "iyaquemetl" = "insect"** → Not attested
11. **Unit 36: "tlatzehtzeloltic" = "holy"** → Not attested. "Holy" = **"tlateochihualli"**
12. **Unit 38: "tecciztli" = "egg"** → Means **"conch shell"**. Egg = **"totoltetl"**

### Major structural/organizational issues:

1. **Country names (Roma, Malta, Rusia, Japón, Suiza, Chile, Palestina, Oman)** scattered throughout unrelated units
2. **Massive orthographic inconsistency** — freely mixes IDIEZ (c/qu), INALI (k/w), and SEP (s/j) without any labeling
3. **Excessive unlabeled duplicates** — many words appear 3-5 times in different spellings without indicating which orthographic system each represents
4. **Classical Nahuatl forms mixed with modern EHN** without distinction
5. **Spanish loanwords not labeled** as such (médico, soldado, mesa, siya, puerta, amigo, gobierno, escuela, etc.)

### Recommendations:

1. **Pick one orthographic system** (ideally IDIEZ or INALI) and use it consistently. If including variants, label them clearly
2. **Remove or re-categorize** the misplaced country names
3. **Correct all 12 critical definition errors** listed above
4. **Label Spanish loanwords** with a marker (e.g., "Sp." or "loanword")
5. **Add pronunciation guides** for community-specific variants (Chicontepec vs. Huejutla, etc.)
6. **Cross-reference against IDIEZ dictionary** (University of Oregon Nahuatl Dictionary, IDIEZ module) for final verification of all entries
